# Haoyu Zhao Homepage

This repository contains the source for Haoyu Zhao's personal academic homepage, built with Jekyll and deployed to GitHub Pages.

## Local preview

1. Install Ruby, Bundler, and Node.js.
2. Run `bundle install`.
3. Run `npm install`.
4. Run `bundle exec jekyll serve -l -H localhost`.

If you change files under `assets/js/`, rebuild the minified bundle with `npm run build:js`.
